# AT-Lab

Необходимо создать приложение, которое будет считывать текстовые данные:
1.	Из указанного файла.
2.	Либо всех файлов в указанной папке, если расширение файла поддерживается программой. Необходим список поддерживыемых расширений.
и выполнять следующие действия:
•	Выводить на экран информацию о файле. Например: {file_name} | {file_ext} | {size}
•	С использованием регулярных выражений проверять в тексте наличие русских символов и, если таковые будут, отобразить на экране каждую подстроку с 
указанием номера строки и номера символа. Например:  Ln: 1    Col: 6    Text: сор
•	В случае, если встреченную подстроку можно изменить на соответсвующую английскую, заменить ее и информацию об этом отобразить в строке вывода, описанной выше.
•	В конце работы в файлом указать количество найденных русских подстрок и количество возможных исправлений.
•	Спросить у пользователя разрешение на сохранение измененных текстовых данных. Если пользователь подтверждает сохранение: сохранить новые данные под 
оригинальным именем, а старый файл сохранить переместить в папку Backup.
